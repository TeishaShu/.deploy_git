TypeScript5- 函式、void、never

發表於 2022-08-05 更新於 2024-02-21 分類於 JavaScript

void、never 主要使用在函式的返回值。

函式

  • TypeScript 中「=>」的左邊式輸入的型別需要用「()」括起來，右邊是返回值的型別

  • 基本寫法

  • ES6 中「=>」是箭頭函式

void

表示: 空的，沒有返回值

never

  • 表示: 沒有值，永遠不會回傳結果。
  • 出錯使用，連 undefined 也沒有
  • 比較少使用
